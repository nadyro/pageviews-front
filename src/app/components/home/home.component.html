What is Pageviews ?

Pageviews is a website that allows you to peek an eye on Wikipedia pages views across the world.

How does it work ?

This form down here allows you to select a date at which you want to browse data from Wikipedia pages.

You also have to select an hour of the day you chose.

Once you're all set, click 'Submit' to trigger the Wikipedia pages computing.

Okay, what's next ?

Next, if it is the first time you ask for a date, please allow few seconds for the system to process the data it fetches.

Once it is done, you will see a Sub-domains selector under the form.

There, select all the sub-domains you want to check.
